classmethod makeYawPitch(vec, order=0)¶

球面上のベクトルからヨーとピッチの2軸回転を得る。

基準軸を指定方向へ向けるヨー、ピッチの回転が得られる。

回転オーダーは、そのまま 初期方向を表す基準軸、ピッチ回転軸、ヨー回転軸 を表す。
